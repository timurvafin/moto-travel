# Moto Travel

Шаблоны для планирования мото-путешествий с помощью [Claude Code](https://claude.ai/claude-code).

Каждая поездка — отдельный каталог в `trips/` с маршрутом, чеклистом и бюджетом в Markdown.

## Структура

```
moto-travel/
├── CLAUDE.md                 # Инструкции для Claude Code
├── trip-template.md          # Шаблон маршрута
├── checklist-template.md     # Шаблон чеклиста
├── budget-template.md        # Шаблон бюджета
└── trips/
    └── <год-название>/
        ├── <год-название>-v<N>.md        # Маршрут, POI
        ├── <год-название>-checklist.md   # Чеклист
        └── <год-название>-budget.md      # Бюджет
```

## Быстрый старт

1. Склонируйте репозиторий
2. Откройте в Claude Code
3. Попросите создать новую поездку:
   ```
   Создай поездку 2026-altai на 10 дней, старт из Новосибирска
   ```

Claude Code использует `CLAUDE.md` как набор правил: учитывает характеристики мотоциклов, планирует заправки, проверяет расстояния и собирает информацию из мото-форумов.

## Шаблоны

- **trip-template.md** — маршрут по дням с точками интереса, расстояниями и временем
- **checklist-template.md** — подготовка к поездке и список снаряжения
- **budget-template.md** — расходы, расчёт бензина, жильё

## Настройка под себя

Отредактируйте `CLAUDE.md`:
- Укажите свои мотоциклы (бак, расход, вес)
- Настройте правила планирования (дневной пробег, время выезда)
- Добавьте предпочитаемые источники информации
